### 3️⃣ Deploy (1 hora)

Escolha uma plataforma:

1. **Netlify Drop** (Mais fácil - 1 min)
   - Arraste pasta para https://app.netlify.com/drop
   
2. **Vercel** (Rápido - 3 min)
   - `npm install -g vercel` → `vercel`

3. **GitHub Pages** (Gratuito - 5 min)
   - Push para repositório
   
4. **Seu próprio servidor** (Controle total)
   - Veja [DEPLOYMENT.md](DEPLOYMENT.md)

**Guia detalhado**: Veja [DEPLOYMENT.md](DEPLOYMENT.md)

## 🆘 Se Algo der Errado

### Página não carrega

✓ Verifique console (F12 > Console)
✓ Teste arquivo local vs servidor
✓ Limpe cache (Ctrl+Shift+Delete)

### Estilos não aparecem

✓ CSS está no mesmo diretório?
✓ Verifique path no HTML
✓ Sem caracteres especiais nos nomes

### JavaScript não funciona

✓ Console sem erros?
✓ JavaScript habilitado no navegador?
✓ Servidor rodando localmente?

### Imagens não aparecem

✓ URL correta?
✓ Arquivo existe?
✓ Sem erro 404 no console?
